Class: responsive/router

responsive/router

new responsive/router()

an object responsible for sending events to statefulviews on the page. whenever the route changes, the router checks each view to see if the route matches the any of the view's routes if so, the views showState method is called, otherwise lostState is called. should not be instantiated directly.
