isComplete()

isComplete is called by router.js. used to determine if the widget is actualy complete. override if you are using an ajax call to populate data. once all stateful views on a page respond with true, the page is considered complete, and router.js sends a 'routecomplete' event indicating so. page.js listens for this event... is very helpful for making ajax applications crawlable by search engines

lostState(route, params)

the router has determined that this view should not be active. hide the contents of this view. the dom node is actually removed from the dom and kept in memory for later insertion so search engines are not tracking non-relevant content

showState: function(route, params) {
	
	if (this.checkLastState(params)) {
		var self = this;
		this.listRendered = false;
		model.findAll(params, function(items) {
			self.showItems(items);
			self.listRendered = true;
		});
	}
	this.inherited(arguments);
}
